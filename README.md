# 📊 Bayesian Statistics – Gun Control Support Estimation

This project demonstrates a simple Bayesian approach to estimate public support for gun control using a Beta prior and survey data. The model updates beliefs based on observed responses and produces a posterior distribution with a credible interval.

---
## 🛠️ Tools & Technologies
- Python
- NumPy
- SciPy
- Matplotlib

---

## ✨ Key Features
- Bayesian updating using Beta-Binomial model  
- Prior → Likelihood → Posterior workflow  
- Credible interval estimation  
- Clear visualization of prior vs posterior beliefs  
- Simple and interpretable statistical model  

---

## 🧭 Workflow
- Defined a Beta prior distribution to represent initial belief  
- Collected survey results (support vs non-support)  
- Updated beliefs using Bayesian inference  
- Computed posterior distribution  
- Extracted credible interval  
- Visualized results for interpretation  

---

## 💡 What I Learned
- How Bayesian inference updates prior beliefs with data  
- Relationship between Beta distribution and binomial outcomes  
- How to interpret posterior distributions in real-world problems  
- Importance of uncertainty quantification (credible intervals)  
- Basics of probabilistic modeling in Python
